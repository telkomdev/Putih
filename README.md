## Putih

Redis Command Line Client Writen in `C++`

### Requirements
- gcc https://gcc.gnu.org/install/binaries.html
- cmake https://cmake.org/download/

# Building

Change directory to root project
```shell
$ cd putih
```

Create `build` folder
```shell
$ mkdir build
```

Run `cmake`
```shell
$ cmake ..
```

Run `make`
```shell
$ make
```
